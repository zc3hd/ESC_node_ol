```
【思路】

1.把需要上传的文件全部放在一个文件夹下
2.流出任务列表，上传文件名称，上传到哪的配置。

  {
    src: './files_to_ol/index.js',
    to: 'db_from_out/',
  },
  {
    src: './files_to_ol/tool.js',

  },
```
